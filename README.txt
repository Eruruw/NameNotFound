Needed Dependencies


NodeJS: Type into google "download nodejs" and go to their website to download the prebuilt installer for windows

React: npx create-react-app <new folder to drop the react stuff in>

React-Router-Dom: npm install react-router-dom

React-Bootstrap: npm install bootstrap react-bootstrap

Firebase: npm install firebase

For the google api: npm install @react-google-maps/api

**Make sure you are in the directory with react to download all of this stuff so you're whole project has access to it**
